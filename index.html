Motivation

So why would Zach want to be a data engineer?

Well, I had enough inherited wealth to pretty much drop out of the sytem for four years, and could go for a lot longer at my current burn rate. (actually I have more money now than I did four years ago but I'm all cash at this point, and going forward I think equity return is going to be like 3%).

However from the perspective of 99% of people I don't have FU money. I'm still relatively young (26) and would like to experiment a bit with more lifestyle changes since I've pretty thoroughly explored the cheap end of the spectrum (quite expert at building standing desks out of boxes or library books, tried sleeping in a hammock, ate diy soylent for a couple years (like $3 a day)). Think hiring a personal trainer, having health insurance (lol), maybe braces, living next to a hackerspace instead of with my parents, etc.

Also, equally importantly, it's pretty clear at this point that while the project is possible it will take quite a while (at least without vc which I don't want, and in fact I would prefer to run the business as a lifestyle business, assuming it succeeds). If I had a couple years of salary saved up I could afford to pay more people to scale up the business/take it off my hands without bankrupting myself, so that would be nice.

Also I've been to a lot of meetups. Dar seems like a good guy. So this is only the second time I've a applied for a position in the past 4 years. Plus it helped motivate me to get my site deployed lol. I think data science is interesting if somewhat scary (sometimes I browse the the arxiv, cs new is like 90% neural nets lol). Haven't done much myself as far as machine learning or stats is concerned on account of trying to be linearly independent from the rest of society so I can do my best work as a living gpu core.
